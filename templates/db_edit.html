使い方メモ

■表示関係

  • 全てのタスクIDを表示する:
    • SELECT id FROM task;
  • IDが4のタスクの全項目を表示する:
    • SELECT * FROM task WHERE id = 4;

■内容の書き換え

  • 特定のタスクの項目を更新する:
    • UPDATE task SET column_name = 'new_value' WHERE id = 4;

■データの削除

  • 特定のタスクを削除する:
    • DELETE FROM task WHERE id = 4;
